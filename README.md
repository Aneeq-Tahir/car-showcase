# Car Hub

## Description

Car Hub is an exhibition website that showcases various car models and provides detailed information about each one. The website is developed using Next.js and features car images and information sourced from Rapid API's APIs.
It also offers dark mode support through the 'next-themes' library. The design is crafted with Shadcn UI and Tailwind CSS.
Go to [Car Hub](https://car-showcase-aneeq-tahir.vercel.app) to visit the website.
